# 사라진 시간의 방

플레이어가 직접 사물을 조사하고 단서를 조합해 탈출하는 1시간 분량의 웹 방탈출 게임입니다. 기존처럼 정답과 풀이 흐름을 먼저 보여주는 안내 화면이 아니라, 실제 플레이 감각에 맞춘 탐색형 구조로 구성했습니다.

## 주요 기능

- 60분 카운트다운 타이머
- 입구 / 서재 / 갤러리 / 시간의 문 구역 탐색
- 사물 조사 버튼과 조사 기록 저널
- 퍼즐별 점진적 힌트 공개
- 퍼즐 4단계 순차 해금
- 최종 탈출 성공 / 시간 초과 엔딩
- Windows에서 바로 실행 가능한 `run-server.bat` 포함

## 실행 방법

### 빠른 실행

```bash
npm run dev
```

기본 주소는 `http://localhost:4173` 입니다.

### Windows 배치 파일 실행

1. Node.js LTS를 설치합니다.
2. 프로젝트 루트에서 `run-server.bat`를 실행합니다.
3. 배치 파일은 Windows 콘솔 인코딩 문제를 피하기 위해 ASCII 메시지만 출력합니다.
4. 브라우저에서 `http://localhost:4173` 으로 접속합니다.

## 게임 플레이 방식

1. **게임 시작**을 눌러 타이머를 시작합니다.
2. 각 구역의 사물을 눌러 조사 기록을 모읍니다.
3. 퍼즐 카드에는 이미 찾은 단서만 표시됩니다.
4. 막히면 **힌트 보기**로 힌트를 한 단계씩 공개할 수 있습니다.
5. 마지막에 시계, 서재, 그림 퍼즐 결과를 순서대로 합쳐 시간의 문을 엽니다.

## 퍼즐 구성

- **시간의 시계**: 입구에서 시간 흐름을 찾는 퍼즐
- **과거의 서재**: 책장과 페이지 단서를 조합하는 퍼즐
- **미래의 그림**: 편지 조각과 프레임 문장을 결합하는 퍼즐
- **시간의 문**: 앞선 세 결과를 하나로 결합하는 최종 퍼즐
