# 사라진 시간의 방

1시간 분량의 방탈출 웹앱입니다. 플레이어는 네 개의 핵심 퍼즐을 차례대로 풀어 `시간의 문`을 열어야 합니다.

## 주요 기능

- 60분 카운트다운 타이머
- 스토리, 단계별 진행표, 타이머 기반 힌트 제공
- 4단계 퍼즐 진행 및 순차 해금
- 인벤토리 기반 단서 수집 UI
- 최종 탈출 성공 / 시간 초과 엔딩 제공
- Windows에서 바로 실행 가능한 `run-server.bat` 포함
- 별도 패키지 설치 없이 Node.js만으로 구동 가능

## 실행 방법

### 빠른 실행

```bash
npm run dev
```

기본 주소는 `http://localhost:4173` 입니다.

### Windows 배치 파일 실행

1. Node.js LTS를 설치합니다.
2. 프로젝트 루트에서 `run-server.bat`를 실행합니다.
3. 브라우저에서 `http://localhost:4173` 으로 접속합니다.

## 게임 구성

### 퍼즐 1 · 시간의 시계
- 책에 숨겨진 시간과 벽 시계를 비교해 숫자 순서를 찾습니다.
- 보상: `1-3-4-2`

### 퍼즐 2 · 과거의 서재
- 책장 열/페이지/표지 색 조합으로 상자 위치를 찾습니다.
- 보상: `B-7`, 편지 조각

### 퍼즐 3 · 미래의 그림
- 편지 조각과 그림 속 문구를 결합해 방향 암호를 해독합니다.
- 보상: `좌-우-위-아래`

### 최종 퍼즐 · 시간의 문
- 앞선 세 단서를 `1-3-4-2+B-7+좌-우-위-아래` 형태로 조합합니다.
