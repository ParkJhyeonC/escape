# 학생맞춤통합지원 웹앱 실행 방법

## 1) 실행
1. `start_webapp.bat` 파일을 더블클릭합니다.
2. 검은 콘솔창이 열리고 서버가 실행됩니다.
3. **이 창을 닫지 마세요.** 창을 닫으면 서버도 함께 종료됩니다.
4. 서버가 실행되면 콘솔에 접속 주소가 표시됩니다.
   - 예: `http://localhost:8000`
   - 예: `http://192.168.0.15:8000` (같은 인트라넷 사용자 접속)

## 2) 다른 교직원 접속
- 서버를 실행한 PC와 **같은 학교 네트워크**에 있는 기기에서
- 콘솔에 나온 `http://내IP:포트` 주소를 브라우저에 입력하면 됩니다.

## 3) 포트 변경(선택)
- 명령 프롬프트에서 아래처럼 실행하면 포트를 바꿀 수 있습니다.

```bat
start_webapp.bat 8080
```

## 4) 바로 꺼지는 경우 점검
- Python이 설치되어 있는지 확인합니다.
- 다른 프로그램이 같은 포트를 쓰는지 확인하고 포트를 바꿔 실행합니다.
  - 예: `start_webapp.bat 8080`
- 스크립트는 오류 시 자동으로 `pause`되어 메시지를 확인할 수 있습니다.

## 5) 관리자 비밀번호
- 초기 비밀번호: `1234`
- 관리자 로그인 후 비밀번호를 변경할 수 있습니다.
- 현재 MVP 버전은 비밀번호를 브라우저 `localStorage`에 저장합니다.
