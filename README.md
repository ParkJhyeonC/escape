# 학생맞춤통합지원 웹앱 실행 방법

## 실행
1. `start_webapp.bat`를 더블클릭합니다.
2. 콘솔에 접속 주소가 표시되면 브라우저로 접속합니다.
3. 같은 인트라넷 사용자는 `http://서버PC_IP:포트`로 접속할 수 있습니다.

## 로그인 구조
- 메인 페이지 전에 **교직원 로그인 페이지**가 먼저 열립니다.
- 교직원 로그인은 `선생님 성함 + 비밀번호`로 진행합니다.
- 선생님 계정은 관리자가 여러 이름을 한 번에 등록할 수 있습니다(한 줄에 1명).
- 초기 비밀번호는 `1234`이며 최초 로그인 시 비밀번호를 변경하면서 `확인 답안`을 함께 설정합니다.
- 로그인 후 직책 정보를 입력합니다: `담임교사`(학년/반 입력) 또는 `부장교사`(부서 입력).
- 비밀번호 찾기는 본인이 설정한 `확인 답안`으로 재설정할 수 있습니다.

## 핵심 기능
- 교직원:
  - 로그인 정보(선생님 성함)가 제보 입력에 자동 반영
  - 사례번호 접속 후 부서 지원방향 작성/공유
  - **본인이 작성한 지원방향만** 수정/삭제 가능
- 관리자 대시보드:
  - 탭 분리: 신규제보 접수함 / 사례관리 / 교직원 계정관리
  - 선생님 계정 일괄 등록(여러 이름, 초기 비밀번호 1234)
  - 신규 제보 삭제
  - 제보 기반 사례번호 생성
  - 사례번호 코드 변경(SI → 원하는 코드)
  - 필요 시 수동 사례번호 직접 지정
  - 사례 상태(지원 진행/점검/종결) 변경
  - 사례 삭제
  - 관리자 비밀번호 변경(초기값 1234)

## 데이터 저장
- 서버 파일 `data.json`에 저장됩니다.
- 같은 서버에 접속한 사용자끼리 사례 데이터가 공유됩니다.

## 방화벽 자동 허용
- `start_webapp.bat` 실행 시 Windows 방화벽 인바운드 규칙(TCP 포트)을 자동으로 등록하려고 시도합니다.
- UAC(관리자 권한) 팝업이 뜨면 허용해야 외부 PC 접속(`http://서버PC_IP:포트`)이 가능합니다.
- 팝업을 취소하면 서버는 실행되지만, 다른 PC에서 접속이 계속 차단될 수 있습니다.

수동으로 열어야 할 때(관리자 CMD):
```bat
netsh advfirewall firewall add rule name="StudentSupportWebApp_8000" dir=in action=allow protocol=TCP localport=8000 profile=private,domain
```

## 포트 변경
```bat
start_webapp.bat 8080
```
